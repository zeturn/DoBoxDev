## 💡 架构设计

### 后端架构
```
cmd/          - 应用入口
internal/     - 内部包
  ├── config/    - 配置管理
  ├── models/    - 数据模型
  ├── handlers/  - HTTP 处理器
  ├── middleware/ - 中间件
  ├── database/  - 数据库
  ├── docker/    - Docker 服务
  └── services/  - 业务逻辑
pkg/          - 公共包
  └── utils/    - 工具函数
```

### 前端架构
```
src/
  ├── components/  - 可复用组件
  ├── pages/       - 页面组件
  ├── services/    - API 服务
  ├── hooks/       - 自定义 Hooks
  ├── types/       - TypeScript 类型
  └── utils/       - 工具函数
```
